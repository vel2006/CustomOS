# CustomOS

## Warning
This is currently within development, DO NOT USE THIS DISTROBUTION!

### Abstract

CustomOS is a flavor of Linux designed to be obscure and secure by default. Having several security focused changes to not only the Linux kernel but to the userspace and file system.

### Timeline

This project is expected to take several years to complete. A full timeline is not feasable at the current moment due to the longevity of this project, a simpler and less detailed one is below.

      File System
      Full disk encryption
      File level encryption
      Full RAM encryption via kernel
      Customized GNOME session
      Live ISO
      Installer image for a full disk install
